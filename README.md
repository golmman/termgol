# termgol

## Features

- pause and drawing
- predefined cell setups
- delay
- frames per second
- rules

## Ideas

- fading cells
- screensaver mode
- toroidal or sheet world
- cell setups from files

## Nice Settings

```
cargo run --release -- -c acorn -r B2/S
cargo run --release -- -c termgol -r B3/S012345678
cargo run --release -- -c termgol -r B36/S125
```
